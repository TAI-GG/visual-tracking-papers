## Detailed Papers by Topic / 按主题分组的论文详情

### Topic 1: Multi-Object Tracking (MOT) / 多目标跟踪

#### 1. FEMOT: Multi-Object Tracking using Frame and Event Cameras

**arXiv**: [2606.14094](https://arxiv.org/abs/2606.14094) | **Date**: 2026-06-12
**Authors**: Shiao Wang, Xiao Wang, Chao Wang, Yitao Li, Menghao Liu, Bo Jiang, Yaowei Wang, Yonghong Tian, Jin Tang

[EN] Proposes FEMOT, a large-scale RGB-event MOT dataset covering diverse real-world scenarios and 14 challenging attributes. A multimodal tracking framework FEMOTR decouples RGB and event features and fuses them in the frequency domain for robust object localization and identity association.

[CN] 提出FEMOT，一个大规模RGB-事件多目标跟踪数据集，涵盖多种真实场景和14种挑战性属性。多模态跟踪框架FEMOTR解耦RGB和事件特征并在频域中融合，实现鲁棒的目标定位和身份关联。

#### 2. SAM-Deep-EIoU: Selective Mask Propagation for Multi-Object Tracking

**arXiv**: [2606.13033](https://arxiv.org/abs/2606.13033) | **Date**: 2026-06-11
**Authors**: Alexander Holmberg

[EN] Proposes a training-free algorithm that dispatches from a base tracker to a VOS model (SAM3) only when an assignment-uncertainty signal fires. On SportsMOT, achieves state-of-the-art 86.8 HOTA with global track association.

[CN] 提出无需训练的算法，仅当分配不确定性信号触发时才调度VOS模型（SAM3）。在SportsMOT上以86.8 HOTA达到SOTA。

#### 3. HDST-GNN: Heterogeneous Dynamic Spatiotemporal GNN for MOT in UAV

**arXiv**: [2606.05587](https://arxiv.org/abs/2606.05587) | **Date**: 2026-06-04
**Authors**: Phillip Jiang

[EN] Introduces altitude-adaptive edge construction, heterogeneous node representation, and occlusion-gated temporal aggregation for UAV MOT. Achieves 94.51% MOTA and 97.24% IDF1 on VisDrone2019-MOT, reducing identity switches by 81% vs. SORT.

[CN] 引入高度自适应边构建、异构节点表示和遮挡门控时域聚合用于无人机MOT。在VisDrone2019-MOT上达到94.51% MOTA和97.24% IDF1，身份切换减少81%。

#### 4. Seg2Track++: Probabilistic Track Validation for MOTS

**arXiv**: [2606.03875](https://arxiv.org/abs/2606.03875) | **Date**: 2026-06-02
**Authors**: Diogo Mendonça, Tiago Barros, Cristiano Premebida, Urbano J. Nunes

[EN] Integrates SAM2 instance segmentation with a novel track management module for zero-shot MOTS. Uses Bernoulli filter-based Probabilistic Track Validation (PTV) to suppress ghost tracks.

[CN] 将SAM2实例分割与新型轨迹管理模块结合，实现零样本MOTS。使用基于伯努利滤波器的概率轨迹验证（PTV）抑制幽灵轨迹。

#### 5. CANMOT: Class-Aware Noise Modeling for MOT in Autonomous Driving (IROS 2026)

**arXiv**: [2606.03590](https://arxiv.org/abs/2606.03590) | **Date**: 2026-06-02
**Authors**: Timo Osterburg, Stefan Schütte, Torsten Bertram

[EN] Proposes class-aware and object-aligned noise modeling for KF-based 3D MOT. Reveals severe overconfidence in standard KF baselines via ANEES analysis.

[CN] 提出面向KF-3D MOT的类别感知和目标对齐噪声建模。通过ANEES分析揭示标准KF基线的严重过度自信问题。
