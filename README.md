### Topic 1: Transformer-Based Tracking / 基于Transformer的跟踪

#### 1. SENTRY: SAM2-Enhanced Neighbor-Aware and Temporally Reasoned Memory for Visual Tracking (ECCV 2026)

**arXiv**: [2606.24449](https://arxiv.org/abs/2606.24449) | **Date**: 2026-06-23
**Authors**: Mohamad Alansari, Yonathan Michael, Hasan AlMarzouqi, Muzammal Naseer, Naoufel Werghi, Sajid Javed

[EN] This paper re-examines memory updating in SAM2-based visual tracking, discovering that relying solely on confidence scores for mask selection causes drift during occlusions, fast movements, and when distractors appear. The team presents SENTRY, a training-free "refine-before-write" module that verifies memory updates using neighbor-aware cycle-consistent matching. It collects multiple segmentation candidates per frame, traces them into short tracklets, and applies cycle-consistency to prioritize geometrically and temporally aligned masks. When added to five baseline models, SENTRY improves performance across nine benchmarks, setting new zero-shot state-of-the-art on LaSOT and GOT-10k while maintaining ~33 FPS with minimal memory overhead.

[CN] 本文重新审视了基于SAM2的视觉跟踪中的记忆更新机制，发现仅依赖置信度分数进行掩码选择会在遮挡、快速运动和干扰物出现时导致漂移。研究团队提出SENTRY，一个无需训练的"先验证再写入"模块，使用邻域感知循环一致性匹配来验证记忆更新。该模块在每帧收集多个分割候选，追溯为短轨迹片段，并应用循环一致性来优先选择几何和时序对齐的掩码。当添加到五个基线模型时，SENTRY在九个基准上提升了性能，在LaSOT和GOT-10k上创造了新的零样本SOTA，同时保持约33 FPS和极低内存开销。

#### 2. TGTrack: Temporal Generative Learning for Unified Single Object Tracking (CVPR 2026)

**Source**: CVPR 2026 | **Date**: 2026-06
**Authors**: Wanting Geng, Xin Chen, Chuanyu Sun, Jie Zhao, Ben Kang, Dong Wang, Huchuan Lu

[EN] Current tracking models handle temporal modeling poorly by only sharing basic data between frames without true temporal supervision. TGTrack introduces a generative supervision approach using a transformer-based decoder with autoregressive prediction and gated fusion, allowing the system to predict upcoming visual states using historical context. A temporal token embedding clearly marks frame positions. Testing across eleven datasets and five modalities demonstrates top-tier results, achieving 75.3% AUC on LaSOT.

[CN] 当前跟踪模型仅通过帧间共享基础数据处理时序建模，缺乏真正的时序监督。TGTrack引入生成式监督方法，使用基于Transformer的解码器配合自回归预测和门控融合，使系统能够利用历史上下文预测未来的视觉状态。时域token嵌入明确标记帧位置。在11个数据集和5种模态上的测试展示了顶级结果，在LaSOT上达到75.3% AUC。

#### 3. ETCTrack: An Efficient Token Compression Framework for Visual Object Tracking (CVPR 2026)

**Source**: CVPR 2026 | **Date**: 2026-06
**Authors**: Weijing Wu, Qihua Liang, Bineng Zhong, Haiying Xia, Zhiyi Mo, Shuxiang Song

[EN] Modern Transformer trackers use numerous historical frames to gather spatial-temporal context but generate excessive visual tokens, causing quadratic computational overhead. ETCTrack introduces a "compress-then-interact" framework that efficiently condenses historical template tokens into compact target representations without manual heuristics. An adaptive token compressor dynamically builds concise templates, and a hierarchical interaction encoder enables deep integration with search features. The B224 variant cuts template tokens by 60% and reduces MACs by 21.4% with merely 0.4% accuracy drop across seven benchmarks.

[CN] 现代Transformer跟踪器使用大量历史帧来获取时空上下文，但产生了过多的视觉token，导致二次方计算开销。ETCTrack引入"先压缩再交互"框架，高效地将历史模板token压缩为紧凑的目标表示，无需手动启发式规则。自适应token压缩器动态构建简洁模板，分层交互编码器实现与搜索特征的深度融合。B224变体将模板token减少60%，MACs降低21.4%，在七个基准上仅有0.4%的精度下降。

### Topic 2: Multi-Object Tracking (MOT) / 多目标跟踪

#### 4. Polycepta: Object-Centric Appearance Estimation for Multi-Object Tracking

**arXiv**: [2606.23604](https://arxiv.org/abs/2606.23604) | **Date**: 2026-06-22
**Authors**: Mohamed Nagy, Naoufel Werghi, Jorge Dias, Majid Khonji

[EN] Traditional MOT pipelines depend on fixed visual descriptors that lack robustness and demand heavy computation. Polycepta reformulates visual modeling as a continuous recursive estimation problem, constructing and refreshing independent appearance states for each tracked object. Instead of memorizing static features, the system learns how to build representations, enabling zero-shot class generalization. Visual accuracy improves as more data is gathered during inference. Testing shows fewer identity errors and better overall tracking. The system runs at 90.57 Hz and achieves MOTA of 92.27% on KITTI when paired with RobMOT.

[CN] 传统MOT管线依赖固定视觉描述子，缺乏鲁棒性且计算代价高。Polycepta将视觉建模重新定义为连续递推估计问题，为每个被跟踪对象构建和刷新独立的外观状态。系统不记忆静态特征，而是学习如何构建表示，实现零样本类别泛化。视觉精度随推理过程中数据的积累而持续提升。测试显示更少的身份错误和更好的整体跟踪性能。系统在搭配RobMOT时运行于90.57 Hz，在KITTI上达到92.27% MOTA。

#### 5. STAITUS: Rethinking Object-Centric Representations for Video Dynamics Modeling

**arXiv**: [2606.23436](https://arxiv.org/abs/2606.23436) | **Date**: 2026-06-22
**Authors**: Amaury Wei, Ismail Nejjar, Olga Fink

[EN] In unsupervised video object tracking, slot-based models struggle when appearance and pose are entangled, causing foreground objects to fragment or swap identities. STAITUS explicitly disentangles each slot into appearance and geometric pose (position and scale), enforcing spatial separation within frames and temporal alignment solely in the appearance space. This yields sharper masks and persistent identities during movement, occlusions, and transitions. An adaptive gating mechanism dynamically adjusts the number of active slots to match scene complexity, preventing over-segmentation.

[CN] 在无监督视频目标跟踪中，基于槽的模型在外观和姿态耦合时表现不佳，导致前景对象碎片化或身份切换。STAITUS将每个槽显式解耦为外观和几何姿态（位置和尺度），在帧内强制空间分离，仅在外观空间中进行时序对齐。这产生了更锐利的掩码和在运动、遮挡及转换过程中更持久的身份标识。自适应门控机制动态调整活跃槽数量以匹配场景复杂度，防止过分割。

### Topic 3: Multimodal Tracking / 多模态跟踪

#### 6. APRTrack: Active Adversarial Perturbation-driven Associative Memory Retrieval for RGB-Event Visual Object Tracking

**arXiv**: [2606.26455](https://arxiv.org/abs/2606.26455) | **Date**: 2026-06-24
**Authors**: Xiao Wang, Xufeng Lou, Zikang Yan, Lan Chen, Sibao Chen, Yaowei Wang, Yonghong Tian, Jin Tang

[EN] Combining RGB textures with event-sensor motion cues improves tracking, but real-world conditions frequently degrade these signals via occlusions, truncation, or clutter. APRTrack uses dual adversarial perturbation branches (spatial and modal) to simulate signal corruptions, with a hierarchical routing system to isolate perturbation training and prevent feature collapse. A Footprint-guided Channel-calibrated Hopfield Retrieval (FCHR) module reliably restores missing historical features by evaluating confidence via query-memory associations. Tests across FE108, COESOT, VisEvent, and FELT datasets validate the approach.

[CN] 将RGB纹理与事件传感器运动线索结合可改善跟踪，但真实场景条件经常通过遮挡、截断或杂波降低信号质量。APRTrack使用双重对抗扰动分支（空间和模态）模拟信号退化，配合分层路由系统隔离扰动训练并防止特征崩溃。足迹引导通道校准Hopfield检索（FCHR）模块通过查询-记忆关联评估置信度，可靠地恢复丢失的历史特征。在FE108、COESOT、VisEvent和FELT数据集上的实验验证了该方法的有效性。

#### 7. RAGTrack: Language-aware RGBT Tracking with Retrieval-Augmented Generation (CVPR 2026)

**Source**: CVPR 2026 | **Date**: 2026-06
**Authors**: Hao Li, Yuhao Wang, Wenning Hao, Pingping Zhang, Dong Wang, Huchuan Lu

[EN] Combining thermal and visible spectra helps localize items in varied environments, but prior techniques depended on first-frame visuals alone, struggling with shifting appearances and lacking linguistic direction. RAGTrack incorporates textual descriptions into RGBT tracking using multi-modal large language models for automated label generation. The retrieval-augmented architecture features a Multi-modal Transformer Encoder, an Adaptive Token Fusion mechanism to minimize search space, and a Context-aware Reasoning Module enabling temporal linguistic reasoning. Evaluations across four RGBT benchmarks confirm superior effectiveness.

[CN] 将热红外和可见光谱结合有助于在多种环境中定位目标，但先前技术仅依赖首帧视觉信息，难以应对表观变化且缺乏语言引导。RAGTrack利用多模态大语言模型自动生成文本标签，将文本描述融入RGBT跟踪。检索增强架构包含多模态Transformer编码器、自适应Token融合机制以最小化搜索空间，以及支持时域语言推理的上下文感知推理模块。在四个RGBT基准上的评估确认了其优越性能。

#### 8. 4DVLT: Dynamic Scene Understanding with Worldline-Centered Vision-Language Tracking

**arXiv**: [2606.22631](https://arxiv.org/abs/2606.22631) | **Date**: 2026-06-21
**Authors**: Chaoyue Li, Boxue Yang, Shengyao Zhou, Haoyang Wu, Rui Qian, Linfeng Zhang

[EN] Interpreting 4D environments requires grounding language to persistent trajectories merging identity, 3D movement, and synchronized 2D perspectives. The authors introduce the 4DVLT task and the Instruct-4D benchmark (~129,400 QA pairs, 64,000+ subjects, 851 scenes, 9 reasoning categories). Their 4DTrack model casts instruction-conditioned tracking as graph-conditioned worldline inference through object-centric 4D state graphs, with metric-directed routing, bidirectional decoding, and motion calibration. The 9B parameter model scored 62.68 on the benchmark, beating the top alternative by 19.62 points.

[CN] 理解4D环境需要将语言锚定到融合身份、3D运动和同步2D视角的持久轨迹上。作者提出了4DVLT任务和Instruct-4D基准（约129,400个QA对、64,000+个对象、851个场景、9种推理类别）。4DTrack模型将指令条件跟踪转化为以对象为中心的4D状态图上的图条件世界线推理，包含度量引导路由、双向解码和运动校准。9B参数模型在基准上得分62.68，超越最优方案19.62分。

### Topic 4: 3D Tracking & Point Tracking / 3D跟踪与点跟踪

#### 9. UniKPT: Generalizable Structure-Aware Keypoint Correspondence for Category-Unified 3D Single Object Tracking (CVPR 2026)

**Source**: CVPR 2026 | **Date**: 2026-06
**Authors**: Jie Xiao, Yinchao Ma, Yuyang Tang, Dengqing Yang, Jianpeng Yang, Xu Zhou, Qiao Li, Wenfei Yang, Tianzhu Zhang

[EN] Monitoring individual items in 3D point clouds faces challenges from sparse data and vast structural differences among object categories. Current techniques build distinct algorithms per category, restricting adaptability. UniKPT features a flexible extractor for scale-aware meaningful keypoints, a progressive aligner for cross-frame geometry, and a confidence-based localization unit filtering unreliable matches using structural ties. This unified approach sets new benchmarks on nuScenes and KITTI, even surpassing category-specific algorithms.

[CN] 在3D点云中跟踪单个目标面临数据稀疏和类别间巨大结构差异的挑战。当前技术为每个类别构建不同算法，限制了泛化能力。UniKPT包含尺度感知有意义关键点的灵活提取器、跨帧几何渐进对齐器和基于置信度利用结构关系过滤不可靠匹配的定位单元。这种统一方法在nuScenes和KITTI上创造了新基准，甚至超越了类别特定算法。

#### 10. MV-TAP: Tracking Any Point in Multi-View Videos (CVPR 2026)

**Source**: CVPR 2026 | **Date**: 2026-06
**Authors**: Jahyeok Koo, Ines Hyeonsu Kim, Mungyeom Kim, Junghyun Park, Seohyeon Park, Jaeyeong Kim, Jung Yi, Seokju Cho, Seungryong Kim

[EN] Monocular point tracking suffers from limited geometric information. MV-TAP follows points through dynamic multi-camera footage by exploiting cross-view geometric cues. The system employs camera geometry and cross-view attention mechanisms to merge spatio-temporal signals from various angles, producing more complete and reliable trajectory estimation. A large-scale synthetic training dataset and real-world evaluation sets specifically for multi-view tracking were assembled. Results confirm superiority over prior techniques on challenging benchmarks.

[CN] 单目点跟踪受限于几何信息不足。MV-TAP通过利用跨视角几何线索在动态多相机视频中跟踪点。系统采用相机几何和跨视角注意力机制融合多角度的时空信号，产生更完整可靠的轨迹估计。构建了专门针对多视角跟踪的大规模合成训练数据集和真实世界评估集。结果证实了在挑战性基准上优于先前技术。

#### 11. MVTrack4Gen: Multi-View Point Tracking as Geometric Supervision for 4D Video Generation

**arXiv**: [2606.26087](https://arxiv.org/abs/2606.26087) | **Date**: 2026-06-24
**Authors**: JoungBin Lee, Jaewoo Jung, Jongmin Lee, Tongmin Kim, Hyunsung Kim, Takuya Narihira, Kazumi Fukuda, Jahyeok Koo, Jisang Han, Yuki Mitsufuji, Seungryong Kim

[EN] Generating novel-view videos from a single reference requires maintaining both spatial alignment and motion accuracy. Explicit 3D techniques often fail on moving objects, while camera-conditioned models produce good visuals but poor spatial coherence. MVTrack4Gen presents a training paradigm that uses multi-view point tracking as geometric supervision for diffusion models. Specific attention mechanisms handle spatial matching, and tracking errors directly penalize motion inconsistencies. By routing features to a tracking module and optimizing for point alignment, the method significantly boosts spatial coherence and movement fidelity.

[CN] 从单个参考生成新视角视频需要同时保持空间对齐和运动精度。显式3D技术在运动物体上常失败，相机条件模型产生好的视觉效果但空间一致性差。MVTrack4Gen提出了一种训练范式，使用多视点跟踪作为扩散模型的几何监督。特定注意力机制处理空间匹配，跟踪误差直接惩罚运动不一致。通过将特征路由到跟踪模块并优化点对齐，该方法显著提升了空间一致性和运动保真度。

### Topic 5: Embodied & Video Tracking / 具身与视频跟踪

#### 12. USS: Unified Spatial-Semantic Prompts for Embodied Visual Tracking with Latent Dynamics Learning

**arXiv**: [2606.25880](https://arxiv.org/abs/2606.25880) | **Date**: 2026-06-24
**Authors**: Yuchen Xie, Xinyu Zhou, Kuangji Zuo, Yanshuo Lu, Fengrui Huang, Boyu Ma, Jianfei Yang

[EN] Robotic tracking requires following an object while navigating changing spaces. Current methods relying on text descriptions cause confusion when multiple items match. The USS framework combines physical (spatial) and conceptual (semantic) prompts, processing them with modality-specific encoders and blending with visual data via hybrid attention to output movement waypoints. A latent world model forecasts future states for temporal stability. Physical robot trials show explicit spatial cues outperform text-only approaches, especially with lookalike objects. Virtual trials demonstrate top results and faster speeds than recent large-model approaches.

[CN] 机器人跟踪需要在导航变化空间的同时跟踪目标。当前依赖文本描述的方法在多个目标匹配时产生混淆。USS框架结合物理（空间）和概念（语义）提示，通过模态特定编码器处理并与视觉数据通过混合注意力融合以输出运动路径点。潜在世界模型预测未来状态以保证时序稳定性。真实机器人实验表明显式空间线索优于纯文本方法，特别是在外观相似物体场景中。虚拟实验展示了顶级结果和比近期大模型方法更快的速度。

#### 13. SceneBot: Contact-Prompted General Humanoid Whole Body Tracking with Scene-Interaction

**arXiv**: [2606.27581](https://arxiv.org/abs/2606.27581) | **Date**: 2026-06-25
**Authors**: Sirui Chen, Shibo Zhao, Zhen Wu, Jiaman Li, Guanya Shi, C. Karen Liu

[EN] Existing reinforcement-learning models for humanoids perform well in open areas but fail at tasks requiring heavy physical contact. SceneBot presents a unified framework managing open-area movement, uneven ground navigation, and full-body object manipulation.
