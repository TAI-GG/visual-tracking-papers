# Visual Tracking Papers

A curated collection of recent academic papers on **Visual Object Tracking (VOT)**, updated regularly.

## About

This repository compiles the latest research in visual object tracking, covering topics including:

- **Single Object Tracking (SOT)** — Transformer-based, Siamese, generative, and multimodal approaches
- **Multi-Object Tracking (MOT)** — Detection-based, segmentation-assisted, graph neural network, and edge deployment approaches
- **3D / 4D Tracking** — Point cloud tracking, multi-view tracking, LiDAR-based pedestrian tracking
- **Multimodal Tracking** — RGB-T, RGB-Event, audio-visual, hyperspectral + language-guided tracking
- **Tracking System Deployment** — Edge inference, Kalman filter optimization, real-time performance
- **Datasets & Benchmarks** — New datasets and evaluation protocols for tracking tasks

Papers are sourced from **arXiv (cs.CV)**, **CVPR**, **ECCV**, **ICCV**, **NeurIPS**, and other top venues.
