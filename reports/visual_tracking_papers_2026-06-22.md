#### 1. FEMOT: Multi-Object Tracking using Frame and Event Cameras / FEMOT：基于帧相机与事件相机的多目标跟踪

**arXiv**: [2606.14094](https://arxiv.org/abs/2606.14094) | **Date / 日期**: 2026-06-12  
**Authors / 作者**: Shiao Wang, Xiao Wang, Chao Wang, Yitao Li, Menghao Liu, Bo Jiang, Yaowei Wang, Yonghong Tian, Jin Tang  
**Source / 来源**: arXiv preprint (cs.CV, cs.AI)

**[EN]** Conventional RGB cameras degrade under motion blur, low illumination, and overexposure. Bio-inspired event cameras offer complementary cues with high temporal resolution and dynamic range. This paper proposes FEMOT, a large-scale RGB-event MOT dataset covering diverse real-world scenarios and 14 challenging attributes. A multimodal tracking framework FEMOTR is also proposed, which decouples RGB and event features and fuses them in the frequency domain for robust object localization and identity association. Extensive experiments on FEMOT and DSEC-MOT validate its effectiveness.

**[CN]** 传统RGB相机在运动模糊、低光照和过曝条件下性能下降。受生物启发的事件相机具有高时间分辨率和高动态范围，可提供互补信息。本文提出FEMOT，一个大规模RGB-事件多目标跟踪数据集，涵盖多种真实场景和14种挑战性属性。同时提出多模态跟踪框架FEMOTR，解耦RGB和事件特征并在频域中融合，实现鲁棒的目标定位和身份关联。在FEMOT和DSEC-MOT上的大量实验验证了其有效性。

**Key Contributions / 核心贡献**: Large-scale RGB-event MOT dataset; Frequency-domain multimodal fusion framework

#### 2. SAM-Deep-EIoU: Selective Mask Propagation for Multi-Object Tracking / SAM-Deep-EIoU：面向多目标跟踪的选择性掩膜传播

**arXiv**: [2606.13033](https://arxiv.org/abs/2606.13033) | **Date / 日期**: 2026-06-11  
**Authors / 作者**: Alexander Holmberg  
**Source / 来源**: arXiv preprint (cs.CV)

**[EN]** MOT has a heavy-tailed difficulty distribution: most frames are easy for lightweight base trackers, while a small fraction are intrinsically hard. VOS models can preserve identity through hard frames but are computationally expensive. This paper proposes selective mask propagation — a training-free algorithm that dispatches from a base tracker to a VOS model (SAM3) only when an assignment-uncertainty signal fires, and only modifies base tracker output when VOS makes a confident contradicting prediction. On SportsMOT, SAM3-Deep-EIoU with global track association achieves state-of-the-art 86.8 HOTA.

**[CN]** 多目标跟踪具有重尾难度分布：大多数帧对轻量级基础跟踪器而言很简单，但少数帧本质上很困难。VOS模型能在困难帧中保持身份一致性，但计算开销大。本文提出选择性掩膜传播——一种无需训练的算法，仅当分配不确定性信号触发时才调度VOS模型（SAM3），且仅当VOS做出与基础跟踪器矛盾的置信预测时才修改输出。在SportsMOT上，SAM3-Deep-EIoU结合全局轨迹关联达到SOTA的86.8 HOTA。

**Key Contributions / 核心贡献**: Training-free selective dispatching between base tracker and VOS model; SOTA on SportsMOT

#### 3. HDST-GNN: Heterogeneous Dynamic Spatiotemporal Graph Neural Networks for MOT in UAV Aerial Imagery / HDST-GNN：面向无人机航拍图像多目标跟踪的异构动态时空图神经网络

**arXiv**: [2606.05587](https://arxiv.org/abs/2606.05587) | **Date / 日期**: 2026-06-04  
**Authors / 作者**: Phillip Jiang  
**Source / 来源**: arXiv preprint (cs.CV, cs.AI, cs.LG)

**[EN]** MOT from UAV imagery faces unique challenges: altitude variation, small densely-packed objects, and frequent occlusion causing identity switches. HDST-GNN introduces three contributions: (1) Altitude-Adaptive Edge Construction that adjusts graph connectivity radius from camera-altitude proxy; (2) Heterogeneous Node Representation modeling detections, confirmed tracklets, and lost tracklets as distinct node types; (3) Occlusion-Gated Temporal Aggregation that gates attention by occlusion confidence. On VisDrone2019-MOT, HDST-GNN achieves 94.51% MOTA and 97.24% IDF1, reducing identity switches by 81% vs. SORT.

**[CN]** 无人机航拍图像的多目标跟踪面临独特挑战：高度变化、密集小目标和频繁遮挡导致身份切换。HDST-GNN提出三个创新：(1) 高度自适应边构建，根据相机高度代理调整图连接半径；(2) 异构节点表示，将检测、确认轨迹和丢失轨迹建模为不同节点类型；(3) 遮挡门控时域聚合，根据遮挡置信度门控注意力贡献。在VisDrone2019-MOT上，HDST-GNN达到94.51% MOTA和97.24% IDF1，身份切换相比SORT减少81%。

**Key Contributions / 核心贡献**: Heterogeneous graph modeling for UAV MOT; Altitude-adaptive graph construction; 81% identity switch reduction

#### 4. Seg2Track++: Probabilistic Track Validation and Data Association for MOTS / Seg2Track++：面向多目标跟踪与分割的概率轨迹验证与数据关联

**arXiv**: [2606.03875](https://arxiv.org/abs/2606.03875) | **Date / 日期**: 2026-06-02  
**Authors / 作者**: Diogo Mendonça, Tiago Barros, Cristiano Premebida, Urbano J. Nunes  
**Source / 来源**: arXiv preprint (cs.CV)

**[EN]** Seg2Track++ integrates instance segmentation with SAM2 and a novel track management module for zero-shot MOTS. Tracks are associated using Mask Centroid Distance (MCD) and Confidence-Aware Cost Modulation (CCM), while Probabilistic Track Validation (PTV) employs a Bernoulli filter to validate track existence and suppress ghost tracks. Experiments on KITTI MOTS demonstrate improved identity preservation, reduced false-positive propagation, and robust track management without fine-tuning.

**[CN]** Seg2Track++将SAM2的实例分割与新型轨迹管理模块结合，实现零样本多目标跟踪与分割。使用掩膜质心距离（MCD）和置信感知代价调制（CCM）进行轨迹关联，并通过概率轨迹验证（PTV）采用伯努利滤波器验证轨迹存在性并抑制幽灵轨迹。在KITTI MOTS上的实验表明，无需微调即可改善身份保持、减少假阳性传播并实现鲁棒的轨迹管理。

**Key Contributions / 核心贡献**: Zero-shot MOTS with SAM2; Bernoulli filter-based probabilistic track validation

#### 5. CANMOT: Class-Aware Noise Modeling for MOT in Autonomous Driving / CANMOT：面向自动驾驶多目标跟踪的类别感知噪声建模

**arXiv**: [2606.03590](https://arxiv.org/abs/2606.03590) | **Date / 日期**: 2026-06-02 | **Venue / 会议**: IROS 2026  
**Authors / 作者**: Timo Osterburg, Stefan Schütte, Torsten Bertram  
**Source / 来源**: arXiv preprint (cs.RO), submitted to IROS 2026

**[EN]** Kalman filter-based MOT typically uses globally shared noise covariances across object classes. CANMOT revisits this assumption and proposes class-aware and object-aligned noise modeling for KF-based 3D MOT. Class-specific diagonal process and measurement covariance matrices are introduced, optionally expressed in the object coordinate frame to preserve longitudinal-lateral anisotropy. Experiments on nuScenes show improved tracking performance and substantially reduced identity switches, while ANEES analysis reveals severe overconfidence in standard baselines.

**[CN]** 基于卡尔曼滤波的多目标跟踪通常跨目标类别共享全局噪声协方差。CANMOT重新审视这一假设，提出面向KF-3D MOT的类别感知和目标对齐噪声建模。引入类别特定的对角过程和测量协方差矩阵，可选择在目标坐标系中表示以保持纵横各向异性。在nuScenes上的实验表明跟踪性能提升且身份切换显著减少，ANEES分析揭示标准基线存在严重的过度自信问题。

**Key Contributions / 核心贡献**: Class-aware noise modeling for 3D MOT; Reveals overconfidence in standard KF baselines; IROS 2026

#### 6. VLHTrack: Vision-Language Guided Hyperspectral Object Tracking / VLHTrack：视觉-语言引导的高光谱目标跟踪

**arXiv**: [2606.09167](https://arxiv.org/abs/2606.09167) | **Date / 日期**: 2026-06-08  
**Authors / 作者**: Rui Yao, Yuhong Zhang, Kunyang Sun, Hancheng Zhu, Jiaqi Zhao, Zhiwen Shao, Abdulmotaleb El Saddik  
**Source / 来源**: arXiv preprint (cs.CV)

**[EN]** VLHTrack is a novel hyperspectral vision-language joint tracking framework. It incorporates language priors to address spectral redundancy through a Language-Guided Band Selection Module (LBSM), which leverages LLM descriptions to establish semantic-to-spectral mapping. A Multi-Modal Vision-Language Fusion Module integrates visual and linguistic embeddings. To address target deformation in long-term sequences, a Dynamic Template Update with Mamba (DTUM) module learns inter-frame dependencies via selective state space modeling. Experiments on HOT2023 and HOT2024 demonstrate SOTA performance.

**[CN]** VLHTrack是一个新颖的高光谱视觉-语言联合跟踪框架。通过语言引导波段选择模块（LBSM）引入语言先验来解决光谱冗余问题，利用LLM描述建立语义到光谱的映射。多模态视觉-语言融合模块整合视觉和语言嵌入。为解决长期序列中的目标变形问题，动态模板更新Mamba模块（DTUM）通过选择性状态空间建模学习帧间依赖关系。在HOT2023和HOT2024上的实验展示了SOTA性能。

**Key Contributions / 核心贡献**: First VL-guided hyperspectral tracking framework; LLM-based band selection; Mamba-based dynamic template update

#### 7. AVTrack: Audio-Visual Tracking in Human-centric Complex Scenes / AVTrack：面向以人为中心的复杂场景音视频跟踪

**Authors / 作者**: Yaoting Wang, Yun Zhou, Zipei Zhang, Henghui Ding  
**Source / 来源**: arXiv preprint (cs.CV, cs.AI), ICML 2026

**[EN]** Audio-visual speaker tracking aims to localize and track active speakers by leveraging auditory and visual cues for fine-grained human-centric scene understanding. AVTrack introduces a human-centric audio-visual instance segmentation (AVIS) dataset for dynamic real-world scenarios featuring camera motion, visual occlusions, and position changes. **[CN]** 音视频说话人跟踪旨在通过利用听觉和视觉线索来定位和跟踪活跃说话人，实现细粒度的以人为中心的场景理解。现有数据集局限于简单或同质场景，标注粗糙。AVTrack提出了一个面向动态真实场景的以人为中心的音视频实例分割（AVIS）数据集，包含相机运动、视觉遮挡和位置变化等挑战性条件。评估揭示现有方法的显著性能退化，确立AVTrack作为具有挑战性的基准。

**Key Contributions / 核心贡献**: Human-centric AVIS dataset for complex dynamic scenes; ICML 2026; Challenging benchmark

#### 8. SMAC: Spatial-Modal Joint Modeling for Multimodal Object Tracking / SMAC：面向多模态目标跟踪的空间-模态联合建模

**arXiv**: [2606.03370](https://arxiv.org/abs/2606.03370) | **Date / 日期**: 2026-06-02  
**Authors / 作者**: Meijing Gao, Qitai Sun, Huanyu Sun, Bingxuan Yang, Bingzhou Sun, Xu Chen, Yonghao Yan, Yuxuan Yang  
**Source / 来源**: arXiv preprint (eess.IV)

**[EN]** SMAC proposes a spatial-modal convolution fusion and distillation-prompt-based multimodal MOT framework. A Basic module performs spatial feature extraction and modal interaction via decoupled 3D convolution, while a Mixed module models nonlinear cross-modal correlations through amplitude-phase decomposition. A representation collapse network enables adaptive multimodal fusion with Distillation Prompt Guidance (DPG) and Global Modal Difference Aggregation (GMDA). On the UniRTL dataset, the tracker achieves 63.31 HOTA and 79.21 MOTA on the RNT modality.

**[CN]** SMAC提出了一种空间-模态卷积融合和蒸馏提示的多模态MOT框架。基础模块通过解耦3D卷积执行空间特征提取和模态交互，混合模块通过幅相分解建模非线性跨模态相关性。表示坍塌网络通过蒸馏提示引导（DPG）和全局模态差异聚合（GMDA）实现自适应多模态融合。在UniRTL数据集上，跟踪器在RNT模态上达到63.31 HOTA和79.21 MOTA。

**Key Contributions / 核心贡献**: Spatial-modal fusion backbone with decoupled 3D convolution; Adaptive multimodal fusion via distillation prompts

#### 10. FADE: Out of Sight, Out of Track — Adversarial Attacks on Propagation-based MOT / FADE：眼不见则不跟踪——对基于传播的多目标跟踪器的对抗攻击

**arXiv**: [2604.00452](https://arxiv.org/abs/2604.00452) | **Date / 日期**: 2026-04-01 | **Venue / 会议**: CVPR 2026  
**Authors / 作者**: Halima Bouzidi, Haoyu Liu, et al.  
**Source / 来源**: arXiv preprint (cs.CV), CVPR 2026

**[EN]** FADE is a novel adversarial attack framework exploiting architectural vulnerabilities of Tracking-by-Query-Propagation (TBP) methods. It employs two strategies: (i) Temporal Query Flooding generates spurious temporally consistent track queries to exhaust the tracker's query budget; (ii) Temporal Memory Corruption attacks the query updater's memory via state de-correlation and feature identity erasure. A differentiable pipeline optimizes attacks for physical-world realizability. Experiments on MOT17 and MOT20 show FADE causes significant identity switches and track terminations.

**[CN]** FADE是一个新型对抗攻击框架，利用基于查询传播跟踪（TBP）方法的架构漏洞。采用两种策略：(i) 时域查询洪泛——生成虚假的时域一致轨迹查询以耗尽跟踪器的查询预算；(ii) 时域记忆破坏——通过状态去相关和特征身份擦除攻击查询更新器的记忆。可微流水线优化攻击以实现物理世界可实现性。在MOT17和MOT20上的实验表明FADE导致显著的身份切换和轨迹终止。

**Key Contributions / 核心贡献**: First systematic adversarial attack on TBP trackers; Physical-world realizable attacks; CVPR 2026

#### 11. MV3DT: Fully Distributed Multi-View 3D Tracking in Real-Time / MV3DT：全分布式实时多视角3D跟踪

**arXiv**: [2606.13127](https://arxiv.org/abs/2606.13127) | **Date / 日期**: 2026-06-11  
**Authors / 作者**: Byron Hernandez, Fangyu Li, Aotian Wu, Paul J. Shin, Kaustubh Purandare, Henry Medeiros  
**Source / 来源**: arXiv preprint (cs.CV)

**[EN]** MV3DT is a fully distributed framework for real-time multi-view 3D tracking that achieves accurate identity propagation and occlusion recovery through peer-to-peer coordination, eliminating central aggregation. Each camera node executes a lightweight modular pipeline with monocular 3D perception, distributed multi-view association, and collaborative fusion via lightweight messaging. On WILDTRACK, MV3DT achieves 96.5% IDF1, 93.1% MOTA. It sustains 30 FPS on 100 cameras with <10ms inter-camera latency and only 2.2% communication overhead in a zero-shot regime.

**[CN]** MV3DT是一个全分布式实时多视角3D跟踪框架，通过对等协调实现精确的身份传播和遮挡恢复，消除中心化聚合。每个相机节点执行轻量级模块化流水线，包含单目3D感知、分布式多视角关联和通过轻量消息传递的协作融合。在WILDTRACK上，MV3DT达到96.5% IDF1、93.1% MOTA。在零样本条件下支持100个相机30 FPS，相机间延迟<10ms，通信开销仅2.2%。
